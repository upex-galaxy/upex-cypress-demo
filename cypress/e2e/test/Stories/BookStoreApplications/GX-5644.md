🚩BUSINESS RULES SPEC
About the SUT
The SUT “*BookStore*” was created for testers to practice API testing. It is a bookstore  where, as a user,  you can: register, log in, administrate your own collection of books in your Profile, add books to your collection (from the bookstore), access the detailed information of each book, among other actions.  You CANNOT buy books (it is more like a library than a bookstore)

REGISTER and LOG IN:
✅ You need to be logged in to access your profile (your collection of books is in your Profile)

✅ To register you need: 

-FirstName: REQUIRED  (only UI. NOT required for API testing)

-LastName: REQUIRED  (only UI. NOT required for API testing)

-UserName: REQUIRED

-Password. To create a new user the password must have: 

at least one non alphanumeric character

one digit ('0'-'9')

one uppercase ('A'-'Z')

one lowercase ('a'-'z')

one special character

eight characters or longer

-The “I’m not a robot” captcha must be checked (when registering from the UI. Not applicable from the backend)
