Objetivo del challenge: El objetivo de este challenge es que puedas integrar y aplicar el patrón de diseño de Page Object Model, incluyendo su
creación, uso, exportación e importación. Asimismo, se busca que puedas seleccionar los objetos necesarios para realizar las pruebas de manera
efectiva.

Como aprendiz de control de calidad,Quiero probar: La lista seleccionable se muestra y funciona según lo esperado. La cuadrícula seleccionable se
muestra y funciona según lo esperado. Para que pueda mejorar mis habilidades de prueba para este escenario.

ESPECIFICACIONES DE REGLAS DE NEGOCIO

Para el componente: Las pestañas "Lista" y "Cintura" deben mostrarse de forma predeterminada. La pestaña "Lista" debe estar abierta de forma
predeterminada mostrando la lista de elementos contenida. Comportamiento esperado: Solo se puede mostrar una pestaña a la vez. Los elementos "Lista" y
"Cuadrícula" deben poder seleccionarse y deseleccionarse. Los elementos seleccionados tienen un fondo azul y una fuente blanca. Los elementos no
seleccionados tienen un fondo blanco y una fuente negra.

Para la pestaña "Lista": Elementos de lista predeterminados: Cras justo odio Dapibus ac facilisis ePorta ac consectetur ac Estado predeterminado de
los elementos de la "Lista": No seleccionado Comportamiento esperado: todos los elementos de la lista deben poder seleccionarse o deseleccionarse
según se desee. CUÁNDO: se hace clic en cualquier elemento de la lista Y: no está seleccionado ENTONCES: el elemento en el que se ha hecho clic cambia
a un fondo azul.

Para la pestaña "Cuadrícula": Elementos de cuadrícula predeterminados: Uno Dos Tres Cuatro Cinco Seis Siete Ocho Nueve Visualización esperada:
cuadrícula de 3 x 3. Estado de los elementos esperados: Seleccionado. Comportamiento esperado: los elementos de la lista deben poder anular la
selección y seleccionarse como se desee. CUÁNDO: se hace clic en cualquier elemento de la cuadrícula Y: está seleccionado ENTONCES: el elemento en el
que se ha hecho clic cambia a un fondo blanco.
